About

Assalamu'alaikum warahmatullahi wabarakatuh

Saya lahir di Kab. Bone dan besar di Makassar. Saya berkuliah di Sekolah Tinggi Manajemen Informatika dan Komputer Bandung (STMIK Bandung) jurusan Teknik Informatika dalam jenjang strata 1. sambil kuliah tingkat akhir saya bekerja sebagai Asisten Dosen Mata Kuliah Algo & Pemrograman di STMIK Bandung selama 1 tahun. dan setelah lulus saya kemudian bekerja sebagai Dosen di Sekolah Tinggi Teknologi Indonesia Tanjung Pinang (STTI Tanjung Pinang) Kepulauan Riau selama 4 Tahun. sambil mengajar di STTI Tanjung Pinang saya menerima project skala kecil seperti sistem informasi perpustakaan, penjualan, dan inventori barang berbasis desktop. dan sekarang saya bekerja sebagai guru Produktif jurusan RPL di SMK Negeri 1 Banggai, Kab. Banggai Laut - Sulteng. sambil mengajar di SMK saya pun menerima project skala kecil seperti website sekolah dan wisata berbasis wordpress, serta website sistem informasi.

Saya sangat senang bekerja sebagai seorang programmer karena hobby saya dapat tersalurkan.., karena itu saya sangat senang apabila ada orang yang menawarkan untuk mengerjakan sebuah project sistem informasi berbasi web. berikut keahlian yang saya miliki sekarang :

  • HTML
  • CSS
  • Javascript
  • PHP Native
  • CSS Framework (Bootstrap)
  • PHP Framework (CodeIgniter)
Bagi anda yang ingin membuat sistem informasi berbasis website, ataupun blog pribadi dan instansi silahkan menghubungi saya..
